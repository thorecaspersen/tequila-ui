## Development
Feel free to hack and sumbit a pull request!

### Install development mode
```bash
1. Create config folder in root
2. npm install
```

### Start development mode
```bash
npm run docs
```


### Commit
tequila-ui use commitizen for pull request commits:
```bash
npm run commit
```
